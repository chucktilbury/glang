hash_retv_t find_hash ( hashtable_t *  tab,
const char *  key,
void *  data,
size_t  size

Find the has table entry. If the entry does not exist, then an error is returned and the value that the data parameter points to is undefined. If HASH_NO_ERROR is returned, then the location that the data parameter points to has been filled in with the data that was sored in the table.

Parameters
tab– The table to search.
key– The string to derive the hash from.
data– Pointer to where the data is to be copied to.
size– Number of bytes to copy for the data.
Returns
int – Indicate whether there was an error or not.
